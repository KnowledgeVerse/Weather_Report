# Bihar Meteorological Centre - Monthly Analysis System

## 🌤️ Overview | अवलोकन

This is a comprehensive Weather Monthly Analysis Generator Web Application developed for **Bihar Meteorological Centre (BMC), Patna**. The system automates the generation of monthly weather reports following IMD (India Meteorological Department) standards.

**यह बिहार मौसम विज्ञान केंद्र, पटना के लिए एक व्यापक मासिक मौसम विश्लेषण जनरेटर वेब एप्लिकेशन है। यह प्रणाली भारत मौसम विज्ञान विभाग (IMD) मानकों के अनुसार मासिक मौसम रिपोर्ट तैयार करने को स्वचालित करती है।**

---

## ✨ Key Features | मुख्य विशेषताएं

### 1. **Data Input System | डेटा इनपुट प्रणाली**

- 📁 **Excel Upload**: Supports `.xlsx` files with 3 sheets:
  - `DAILY MAX` - Daily Maximum Temperature
  - `daily min` - Daily Minimum Temperature
  - `RAINFALL` - Daily Rainfall Data
- 📄 **CSV Upload**: Upload 3 separate CSV files
- 🖱️ **Drag & Drop**: Intuitive file upload interface
- ✅ **Auto-validation**: Validates file structure automatically

### 2. **Auto Data Analysis | स्वचालित डेटा विश्लेषण**

The system automatically calculates:

- 🌡️ **Highest Average Maximum Temperature** - सर्वाधिक औसत अधिकतम तापमान
- 🔥 **Highest Single Day Maximum** - सर्वाधिक एकल दिन अधिकतम तापमान
- ❄️ **Lowest Average Minimum Temperature** - सबसे कम औसत न्यूनतम तापमान
- 🧊 **Lowest Single Day Minimum** - सबसे कम एकल दिन न्यूनतम तापमान
- 🏛️ **Patna Specific Statistics** - पटना विशिष्ट आंकड़े
- 🌧️ **Total Monthly Rainfall** - कुल मासिक वर्षा

### 3. **Report Generation | रिपोर्ट जनरेशन**

- 📝 **Hindi Report Text**: Auto-generated in IMD format
- ✏️ **Editable Content**: All text is fully editable
- 🎨 **Customizable Header**: Upload left/right logos
- 📅 **Auto Date**: Current date auto-inserted

### 4. **Chart Generation | चार्ट जनरेशन**

- 📊 **Rainfall Distribution Bar Chart** - जिलों की वर्षा वितरण
- 📈 **5-Year Temperature Trend** - 5 वर्षीय तापमान प्रवृत्ति
- 📉 **5-Year Rainfall Comparison** - 5 वर्षीय वर्षा तुलना
- 💾 **Export Charts**: Download as PNG

### 5. **Monthly Outlook | मासिक दृष्टिकोण**

Editable sections with image upload:

- 🌧️ Rainfall Outlook - वर्षा दृष्टिकोण
- 🌡️ Maximum Temperature Outlook - अधिकतम तापमान दृष्टिकोण
- ❄️ Minimum Temperature Outlook - न्यूनतम तापमान दृष्टिकोण
- ☀️ Heat Wave Outlook - उष्ण लहर दृष्टिकोण

## 🚀 How to Use | उपयोग कैसे करें

### Step 1: Open the Application

Open `index.html` in any modern web browser (Chrome, Firefox, Edge recommended).

### Step 2: Upload Data

**For Excel:**

1. Click on "Excel" tab
2. Drag & drop or click to select `.xlsx` file
3. File must contain sheets: DAILY MAX, daily min, RAINFALL

**For CSV:**

1. Click on "CSV" tab
2. Upload 3 files separately:
   - Daily Max Temperature (daily_max.csv)
   - Daily Min Temperature (daily_min.csv)
   - Rainfall Data (rainfall.csv)

### Step 3: Select Month & Year

- Choose month from dropdown (जनवरी to दिसंबर)
- Select year (2022-2026)

### Step 4: Generate Analysis

Click "विश्लेषण करें" (Analyze) button to:

- Process all data
- Calculate statistics
- Generate charts
- Create report text

### Step 5: Review & Edit

- Review auto-generated statistics
- Edit report text as needed
- Upload outlook maps/images
- Select signatures

### Step 6: Export Report

Click export buttons to download:

- 📕 PDF Report
- 📝 DOCX Report

## 📊 Data Format Requirements | डेटा प्रारूप आवश्यकताएं

### Excel Structure:

```
Sheet 1: DAILY MAX
| Station   | 1   | 2   | 3   | ... | 31  | AVERAGE | max |
|-----------|-----|-----|-----|-----|-----|---------|-----|
| PATNA     | 31.4| 31.0| 31.2| ... | 33.6| 32.6    | 36.2|
| GAYA      | 32.2| 32.0| 31.6| ... | 37.2| 33.7    | 37.8|

Sheet 2: daily min
| Station   | 1   | 2   | 3   | ... | 31  | AVERAGE | min |
|-----------|-----|-----|-----|-----|-----|---------|-----|
| PATNA     | 18.4| 18.2| 18.2| ... | 23.5| 21.5    | 17.6|

Sheet 3: RAINFALL
| Station   | 1   | 2   | 3   | ... | 31  | TOTAL   | max |
|-----------|-----|-----|-----|-----|-----|---------|-----|
| PATNA     | 0   | 0   | 0   | ... | 0   | 16.3    | 7.7 |
```

### CSV Format:

Same structure as Excel sheets, with comma-separated values.
