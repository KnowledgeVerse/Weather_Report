# Weekly Weather Bulletin Generator - Bihar

## 📋 Overview

This is a complete client-side web application that automatically generates Weekly Weather Bulletins for Bihar State in IMD (India Meteorological Department) format.

## ✨ Features

### 1. **Data Input**

- Large textarea for pasting 7 days of daily bulletin text
- Automatic parsing of meteorological data
- Sample data loader for testing

### 2. **Automatic Parsing**

- **Dates**: Extracts and formats dates automatically
- **Weather Systems**: Parses synoptic features and weather patterns
- **Rainfall Data**: Extracts station-wise rainfall amounts
- **Temperature**: Identifies maximum and minimum temperatures
- **Spatial Distribution**: Generates rainfall distribution maps

### 3. **Dynamic Configuration**

- Editable color pickers for rainfall distribution categories:
  - DRY (शुष्क)
  - ISOL (एक या दो स्थानों पर)
  - SCATTERED (कुछ स्थानों पर)
  - FAIRLY WIDESPREAD (अनेक स्थानों पर)
  - WIDESPREAD (अधिकांश स्थानों पर)

### 4. **Preview Sections**

- **Header**: IMD-style header with logos and date range
- **Weather System Summary**: Date-wise synoptic features
- **Rainfall Table**: Station-wise rainfall amounts
- **Spatial Distribution**: Color-coded sector-wise distribution
- **Temperature Scenario**: Highest/Lowest temperature records
- **Temperature Analysis**: Auto-generated Hindi and English summaries
- **Rainfall Map**: Upload and display departure maps

### 5. **PDF Generation**

- One-click PDF generation
- Multi-page layout matching IMD bulletin format
- Professional formatting with proper page breaks

## 🚀 How to Use

### Step 1: Open the Application

Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).

### Step 2: Input Data

1. Paste 7 days of daily bulletin text in the textarea
2. Or click "Load Sample Data" to see a demo
3. Click "Parse Data" to process

### Step 3: Configure Colors

Use the color pickers to set colors for different rainfall distribution categories.

### Step 4: Upload Map (Optional)

Upload the "Rainfall % Departures from Long Period Averages" map image.

### Step 5: Generate PDF

Click "Generate Weekly Bulletin PDF" to download the complete bulletin.

## 📝 Input Format

The system recognizes this format:

```
DATE: 24.03.2026
TIME OF ISSUE: 12:45 HRS IST

पिछले 24 घंटों का राज्य में दर्ज किया गया मौसम सारांश

अधिकतम तापमान: 35.5°C
न्यूनतम तापमान: 16.2°C
हवा का बहाव: पश्चिमी 10-15 kmph

वर्षा की मुख्य मात्रा (मिमी में):
ठाकुरगंज 18.0, पोटठिया 0.8

बिहार राज्य से संबंधित आज की संक्षिप्त मौसम प्रणाली:
➢ Weather system description here...
```

## 🎨 Customization

### Adding Your Logos

1. Replace `images/imd_logo.png` with your IMD logo
2. Replace `images/IMD_150_Year_Logo.png` with your 150 Year logo
3. Recommended size: 80px height

### Modifying Colors

Use the color pickers in the "Rainfall Distribution Configuration" section to customize the spatial distribution table colors.

### Editing Text

All text content can be edited directly in the preview before generating PDF.

## 🔧 Technical Details

- **Pure Client-Side**: No server required, runs entirely in browser
- **Offline Capable**: Works without internet after initial load
- **Libraries Used**:
  - html2canvas (for capturing HTML as images)
  - jsPDF (for PDF generation)
- **Responsive Design**: Works on desktop and mobile devices

## 📄 Output Format

The generated PDF contains:

1. **Page 1**: Header + Weather System Summary
2. **Page 2**: Rainfall Table + Spatial Distribution
3. **Page 3**: Temperature Scenario + Analysis
4. **Page 4**: Rainfall Departure Map (if uploaded)

## ⚠️ Notes

- For best results, use Chrome or Edge browser
- Ensure logo images are properly placed in the `images/` folder
- Large images may take time to process in PDF generation
- All data processing happens locally - no data is sent to any server

## 🆘 Troubleshooting

**PDF not generating?**

- Check browser console for errors
- Ensure pop-ups are not blocked
- Try using Chrome or Edge browser

**Data not parsing correctly?**

- Check date format (DD.MM.YYYY)
- Ensure Hindi keywords are spelled correctly
- Use the sample data as a reference

**Images not showing?**

- Check file paths in `images/` folder
- Ensure images are in PNG or JPG format
- Verify CORS settings if running locally
